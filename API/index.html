Mô Tả API

Đây là API quản lý danh sách sinh viên, được host trên MockAPI. API hỗ trợ đầy đủ các thao tác CRUD — thêm, đọc, cập nhật và xóa — thông qua giao thức HTTP sử dụng định dạng JSON. Không cần xác thực (authentication).

URL gốc: https://69f85af8dd0c226688ee7a09.mockapi.io/Student
Mô hình dữ liệu

Mỗi sinh viên có các trường thông tin sau:

TrườngKiểu dữ liệuMô tả
id string Mã định danh duy nhất, tự động tạo
name string Họ và tên sinh viên
class_name string Tên lớp học của sinh viên
email string Địa chỉ email của sinh viên
mark string Điểm số của sinh viên
Các endpoint
GET /Student
Lấy danh sách tất cả sinh viên. Trả về một mảng (array) các đối tượng sinh viên. 
[
  {
    "id": "1",
    "name": "Nguyễn Văn A",
    "class_name": "Toán",
    "email": "vana@example.com",
    "mark": "88"
  },
  ...
]
GET /Student/:id
Lấy thông tin một sinh viên theo id. Trả về một đối tượng sinh viên duy nhất. 
{
  "id": "1",
  "name": "Nguyễn Văn A",
  "class_name": "Toán",
  "email": "vana@example.com",
  "mark": "88"
}
POST /Student
Tạo mới một sinh viên. Gửi lên một JSON body chứa thông tin sinh viên. Trường id sẽ được tự động gán, không cần truyền vào. 
// Request body
{
  "name": "Trần Thị B",
  "class_name": "Vật Lý",
  "email": "thib@example.com",
  "mark": "75"
}
PUT /Student/:id
Cập nhật thông tin sinh viên theo id. Chỉ cần gửi các trường muốn thay đổi, không cần gửi toàn bộ dữ liệu. 
// Request body (chỉ cập nhật điểm)
{
  "mark": "92"
}
DELETE /Student/:id
Xóa vĩnh viễn sinh viên theo id. Khi thành công, API trả về đối tượng sinh viên vừa bị xóa.
